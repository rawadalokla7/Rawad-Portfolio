## 🚀 How to Deploy to GitHub Pages / خطوات الرفع المباشرة والتشغيل

Follow these simple steps to go live under 1 minute:

### 1. Download Local Assets Automatically
Run the lightweight downloader in your terminal inside this directory to cache high-quality imagery locally:
```bash
node image/download.js
```
This fetches the customized backgroundless optimized PNG (`rawad_portrait.png`) and essential project covers.

### 2. Push directly to your GitHub Repository
Initialize a repository for this directory only and push to main:
```bash
git init
git add .
git commit -m "feat: publish high-fidelity standalone responsive developer portfolio"
git branch -M main
git remote add origin https://github.com/rawadalokla7/rawad-khaled-portfolio.git
git push -u origin main --force
```

### 3. Activate Pages
1. Go to your repository settings on GitHub.
2. Scroll to the **Pages** menu tab on the left bar.
3. Set the source branch to **`main`** and folder directory to **`/` (root)**.
4. Click **Save** and refresh! Your professional live domain is up and running.
